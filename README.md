# Practice CMake

Practice CMake.

- Setup

```bash
# MacOS : Install Using Homebrew
$ brew install cmake
$ cmake --version
cmake version 3.24.2
CMake suite maintained and supported by Kitware (kitware.com/cmake).
```

- Hello World

```bash
$ mkdir hello-world && cd $_
$ touch CMakeLists.txt
$ touch main.cpp

# Create Make Files
$ cmake .

# Build And Execute
$ make
$ hello_world
```
